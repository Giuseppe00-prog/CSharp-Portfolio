# Biblioteca

Applicazione console che gestisce una piccola biblioteca, con funzionalità per aggiungere libri, registrare utenti e tenere traccia dei prestiti.  

## Funzionalità principali

---

- Gestione dei **libri** (aggiunta, ricerca, disponibilità)
- Gestione degli **utenti/lettori**
- **Prestiti** e restituzioni di libri
- Gestione della disponibilità in tempo reale
- Struttura a più classi (Libro, Utente, Prestito, BibliotecaService)
